## Milestone Summary

**Key Decisions:**

- Adopt GSD methodology — Provides structured workflow for AI-assisted development
- Create gsd-workflow skill — Make GSD methodology reusable across projects
- Use Python for helper scripts — Portable, no compilation needed
- Use Contributor Covenant 2.1 for Code of Conduct — Industry standard with 4-tier enforcement
- Use next-i18next over next-intl — Direct integration with Next.js Pages Router built-in i18n routing
- Use native Intl API for date/number/currency formatting — Zero dependencies
- Namespace translations by domain not component — Stable as components are refactored
- triggers: as standard SKILL.md frontmatter field — Consistent with backend-dev-guidelines pattern
- Create INDEX.md at .agent/skills/INDEX.md — Central discovery for all 35+ skills
